Hey, I’m Zack

I’m a Senior Software Engineering Manager working at GitHub

My work focuses on:

  • Leadership: People management, OSPO strategy and execution
  • Customers: Engineering best practices presentations, Hackathons, Conferences, Workshops, Trainings
  • Engineering: Backend web services, GitHub Apps, GitHub Actions
  • Program: Open Source initiatives, InnerSource initiatives, DevSecOps initiatives

What I write about

  • InnerSource - measuring it, scaling it, and proving its value
  • Open source program management - tools and practices for OSPOs
  • DevOps and automation - CI/CD, infrastructure as code, developer tooling
  • GitHub Actions - building and maintaining open source actions

Conference talks

I regularly speak at conferences about InnerSource and open source programs. You can find my recorded talks on my YouTube playlist.

Open source projects

I maintain several open source GitHub Actions in the github-community-projects org:

  • pr-conflict-detector - Catch merge conflicts before they catch you
  • issue-metrics - Metrics for issues, pull requests, and discussions
  • measure-innersource - Measure InnerSource collaboration across your org
  • contributors - Gain insight into your project contributors
  • stale-repos - Find and manage stale repositories
  • evergreen - Keep your dependencies up to date
  • cleanowners - Keep CODEOWNERS files accurate
