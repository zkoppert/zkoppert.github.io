An Open Source Project to Empower OSPOs Everywhere

Posted Mar 13, 2023
By Zack Koppert
1 min read

Running an Open Source Programs Office means juggling a lot of operational tasks: tracking repository health, monitoring contributor activity, ensuring dependency hygiene, and measuring how well your organization collaborates across team boundaries. Most OSPOs end up building custom scripts and dashboards to handle this - reinventing the same wheels that other OSPOs have already built.

We decided to change that by open-sourcing the tools we built at GitHub’s OSPO. This suite of GitHub Actions - including issue-metrics, stale-repos, contributors, evergreen, cleanowners, and measure-innersource - gives any OSPO a ready-made toolkit for the most common operational needs. They’re free, configurable, and designed to work together.
